5. Install and Use Live Server
Install the "Live Server" Extension:
Go to the Extensions panel in VS Code (Ctrl+Shift+X or Cmd+Shift+X).
Search for Live Server and click Install.
Start Live Server:
Right-click on index.html in the VS Code file explorer.
Select Open with Live Server.
Your project will open in your default browser, and any changes you make will refresh automatically.
